# 项目简介

## 核心功能
- 登录注册
- 密码重置
- 个人信息管理
- 交易记录
- 收入和支出记录
- 交易分类管理
- 交易记录搜索和筛选
- 交易记录导出
- 预算管理
- 按分类设置预算
- 预算使用情况追踪
- 预算超支提醒
- 数据统计与分析
- 收支概览
- 收支趋势分析
- 分类统计
- 月度统计
- 数据可视化图表

## 适用场景
- 个人财务管理
- 财务记录与分析
- 预算规划与监控
- 财务报告生成

## 技术特点
- 基于Flask框架，使用MySQL数据库
- 前端采用HTML、CSS和JavaScript
- 提供用户友好的界面设计
- 系统稳定性高，运行稳定
